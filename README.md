# 🏦 Java OOP Bank System

This project is based on Object-Oriented Programming concepts in Java.

## 📌 Features

- Deposit Money
- Withdraw Money
- Balance Check
- Saving Account with Interest
- Current Account with Service Charge

## 🛠 Concepts Used

- Inheritance
- Encapsulation
- Constructors
- Method Overriding
- Access Modifiers

## 📂 Project Structure

OOP2/
│
├── Bank.java
├── BankAccount.java
├── CurrentAccount.java
└── SavingAccount.java

## ▶️ How to Run

Compile:
javac OOP2/*.java

Run:
java OOP2.Bank
